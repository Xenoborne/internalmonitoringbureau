# Internal Monitoring Bureau (IMB)

Unofficial SCP Fan Project – SCP: Roleplay (Roblox)

---

## Overview

The Internal Monitoring Bureau (IMB) is a fan-created SCP Foundation department
designed for use in SCP: Roleplay custom servers on Roblox. This project
provides in-universe documentation, faction lore, and a supporting website for
roleplay, training, and orientation purposes.

This project is not affiliated with the SCP Wiki, the SCP Foundation, Roblox
Corporation, or the developers of SCP: Roleplay.
